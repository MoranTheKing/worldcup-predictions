# Handoff - 2026-04-22

## מצב המוצר

האפליקציה נמצאת כעת באמצע Phase 2 של שכבת המשחק החברתי.

### מה עובד כרגע

- Auth גלובלי לכל האפליקציה
- `/game` כמרכז משחק הניחושים
- ליגות פרטיות עם create / join / leave / remove / delete
- Opponent View מאובטח עם כללי anti-cheat
- Gamer Card עם:
  - `display_name`
  - `total_score`
  - `total_hits`
- בוסטרים נפרדים לשלב הבתים ולנוקאאוט
- UI נעול ואחיד ל־Winner / Top Scorer בעזרת premium badges

## אבני דרך אחרונות

### RTL / Bidi

- תצוגות סקור פוצלו ל־DOM spans נפרדים
- תוצאת הבית מוצמדת לצד ימין
- תוצאת החוץ מוצמדת לצד שמאל
- טופל overflow ותיאום תצוגה בכרטיסי משחק, opponent view ו־dashboard

### Joker

- Exact hit עם Joker עבר לתמת סגול/מג'נטה
- הלוגיקה של Jokers מופרדת בין Group Stage לבין Knockout
- `Total Hits` סופר פגיעות exact בלבד

### פרטיות

- יריבים רואים רק משחקים live/finished
- משחקים scheduled מוסתרים לחלוטין
- placeholders לא פתורים מוסתרים
- outrights של יריבים מוסתרים עד פתיחת הטורניר
- המשתמש עצמו מקבל self-view bypass ורואה/עורך את התוכן שלו

### UI

- LIVE badge בציאן
- Miss או `לא נשלח` במשחק live/finished באדום
- Direction hit בצהוב רך
- Exact hit בירוק
- Exact hit עם Joker בסגול/מג'נטה זוהר

## תיעוד וסנכרון

המסמכים המרכזיים עודכנו כדי לשקף:

- Decoupled RTL score rendering
- Purple Joker theme
- Total Hits
- Strict anti-cheat social viewing
- Unified locked outrights UI
