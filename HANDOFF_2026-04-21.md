## חשוב מאוד למחשב הבא

### 1. למשוך את הקוד
```bash
git checkout master
git pull origin master
```

### 2. להתקין תלויות
```bash
npm install
```

### 3. להחזיר קובץ סביבה
הקובץ `.env.local` לא נשמר ב-Git. צריך להעתיק אותו ידנית למחשב הבא.

נדרשים לפחות:
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`
- `API_FOOTBALL_KEY`

### 4. להריץ migration ב-Supabase
צריך להריץ את תוכן הקובץ:
`supabase/migrations/20260421000005_add_team_standings_cache.sql`

העמוד יעבוד גם בלי זה בגלל fallback, אבל כדי להשתמש בשדות cache האמיתיים של standings צריך להחיל את ה-migration.
